# HU-07: Recordatorios de Turnos

## Historia de Usuario
**Como** paciente  
**Quiero** recibir recordatorios de mis turnos  
**Para** no olvidar mis citas

## Criterios de Aceptación
- [ ] Se envía recordatorio por email 24hs antes del turno
- [ ] Se envía recordatorio por SMS 2hs antes del turno (opcional)
- [ ] El paciente puede configurar preferencias de recordatorios
- [ ] Se incluye información completa del turno en recordatorio
- [ ] Se incluye enlace para cancelar/reprogramar si es necesario
- [ ] Se registra el envío de recordatorios

## Definición de Terminado
- [ ] Sistema de recordatorios automático implementado
- [ ] Configuración de preferencias de usuario
- [ ] Integración con servicio de email
- [ ] Integración con servicio de SMS (opcional)
- [ ] Logs de recordatorios enviados
- [ ] Pruebas automatizadas creadas
