# HU-17: Crear Cuentas de Profesionales y Administrativos

## Historia de Usuario
**Como** master/desarrollador  
**Quiero** crear cuentas de profesionales y administrativos  
**Para** dar acceso al sistema a nuevos usuarios del staff

## Criterios de Aceptación
- [ ] El master puede crear cuentas con rol "professional" o "admin"
- [ ] Se ingresan datos completos del usuario (nombre, email, especialidad para profesionales)
- [ ] Se genera contraseña temporal que debe ser cambiada en primer login
- [ ] Se envían credenciales por email al nuevo usuario
- [ ] Se puede activar/desactivar cuentas existentes
- [ ] Se registra quién creó cada cuenta y cuándo

## Definición de Terminado
- [ ] Interfaz de creación de usuarios master
- [ ] Selección de roles y permisos
- [ ] Generación y envío de credenciales
- [ ] Gestión de estados de cuenta (activo/inactivo)
- [ ] Auditoría de creación de usuarios
- [ ] Validaciones de seguridad implementadas
- [ ] Pruebas unitarias creadas
