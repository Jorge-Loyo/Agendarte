# HU-21: Ver Reseñas al Buscar Profesionales

## Historia de Usuario
**Como** paciente  
**Quiero** ver las reseñas y calificaciones de los profesionales  
**Para** elegir mejor basándome en experiencias de otros pacientes

## Criterios de Aceptación
- [ ] En la lista de profesionales se muestra calificación promedio y número de reseñas
- [ ] Se puede hacer clic para ver todas las reseñas de un profesional
- [ ] Se pueden filtrar profesionales por calificación mínima
- [ ] Se pueden ordenar profesionales por mejor calificación
- [ ] Se muestran reseñas recientes en el perfil del profesional
- [ ] Se respeta el anonimato de las reseñas anónimas

## Definición de Terminado
- [ ] Integración de reseñas en listado de profesionales
- [ ] Vista detallada de reseñas por profesional
- [ ] Filtros por calificación implementados
- [ ] Ordenamiento por rating funcional
- [ ] Respeto de privacidad en reseñas anónimas
- [ ] Interfaz intuitiva para navegación de reseñas
- [ ] Pruebas unitarias creadas
