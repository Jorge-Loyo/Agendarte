# HU-18: Gestionar Permisos y Roles

## Historia de Usuario
**Como** master/desarrollador  
**Quiero** gestionar permisos y roles del sistema  
**Para** controlar el acceso a diferentes funcionalidades

## Criterios de Aceptación
- [ ] El master puede ver todos los usuarios del sistema
- [ ] Se pueden modificar roles de usuarios existentes
- [ ] Se pueden activar/desactivar usuarios
- [ ] Se puede resetear contraseñas de usuarios
- [ ] Se pueden ver logs de acceso y actividad de usuarios
- [ ] Se pueden configurar permisos específicos por rol

## Definición de Terminado
- [ ] Panel de administración de usuarios
- [ ] Gestión completa de roles y permisos
- [ ] Funcionalidades de activación/desactivación
- [ ] Reset de contraseñas
- [ ] Sistema de logs y auditoría
- [ ] Configuración granular de permisos
- [ ] Pruebas unitarias creadas
