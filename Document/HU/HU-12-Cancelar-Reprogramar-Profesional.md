# HU-12: Cancelar/Reprogramar Turnos (Profesional)

## Historia de Usuario
**Como** profesional  
**Quiero** cancelar o reprogramar turnos  
**Para** gestionar cambios en mi agenda

## Criterios de Aceptación
- [ ] El profesional puede cancelar turnos desde su agenda
- [ ] Se puede reprogramar turno seleccionando nueva fecha/hora
- [ ] Se notifica automáticamente al paciente del cambio
- [ ] Se puede agregar motivo de cancelación/reprogramación
- [ ] Se procesa reembolso automático si el turno estaba pago
- [ ] Se registra el cambio en el historial

## Definición de Terminado
- [ ] Funcionalidad de cancelación desde agenda
- [ ] Reprogramación con selector de nueva fecha
- [ ] Notificaciones automáticas al paciente
- [ ] Campo de motivo/observaciones
- [ ] Integración con sistema de reembolsos
- [ ] Registro de cambios en historial
- [ ] Pruebas unitarias creadas
