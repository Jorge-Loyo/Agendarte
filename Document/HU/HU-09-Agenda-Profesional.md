# HU-09: Ver Agenda del Profesional

## Historia de Usuario
**Como** profesional  
**Quiero** ver mi agenda diaria/semanal/mensual  
**Para** organizar mis consultas y tiempo

## Criterios de Aceptación
- [ ] El profesional puede ver agenda en vista diaria, semanal y mensual
- [ ] Se muestran turnos confirmados con datos del paciente
- [ ] Se muestran horarios libres disponibles
- [ ] Se puede hacer clic en turno para ver detalles del paciente
- [ ] Se pueden agregar notas rápidas a los turnos
- [ ] Se puede exportar agenda a PDF o calendario externo

## Definición de Terminado
- [ ] Calendario interactivo implementado
- [ ] Múltiples vistas (día/semana/mes)
- [ ] Información detallada de turnos
- [ ] Funcionalidad de notas
- [ ] Exportación de agenda
- [ ] Interfaz responsive
- [ ] Pruebas unitarias creadas
