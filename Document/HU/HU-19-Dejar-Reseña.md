# HU-19: Dejar Reseña después de Consulta

## Historia de Usuario
**Como** paciente  
**Quiero** dejar una reseña y calificación después de mi consulta  
**Para** ayudar a otros pacientes a elegir profesionales

## Criterios de Aceptación
- [ ] El paciente recibe notificación para reseñar después de consulta completada
- [ ] Se puede calificar de 1 a 5 estrellas
- [ ] Se puede escribir comentario opcional
- [ ] Se puede elegir si la reseña es anónima o no
- [ ] Solo se puede dejar una reseña por turno completado
- [ ] La reseña se publica inmediatamente

## Definición de Terminado
- [ ] Sistema de notificaciones post-consulta
- [ ] Interfaz de calificación y comentarios
- [ ] Opción de anonimato implementada
- [ ] Validación de una reseña por turno
- [ ] Publicación automática de reseñas
- [ ] Integración con perfil del profesional
- [ ] Pruebas unitarias creadas

## Prioridad: Media
## Estimación: 8 puntos
