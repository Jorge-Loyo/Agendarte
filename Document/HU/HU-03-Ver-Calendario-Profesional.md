# HU-03: Ver Calendario de Profesional

## Historia de Usuario
**Como** paciente  
**Quiero** ver el calendario de un profesional  
**Para** elegir horario disponible para mi turno

## Criterios de Aceptación
- [ ] El paciente puede ver calendario mensual del profesional
- [ ] Se muestran horarios disponibles en verde
- [ ] Se muestran horarios ocupados en rojo
- [ ] Se puede navegar entre meses
- [ ] Al hacer clic en horario disponible, se puede agendar
- [ ] Se muestra duración de la consulta

## Definición de Terminado
- [ ] Calendario interactivo implementado
- [ ] Estados visuales claros (disponible/ocupado)
- [ ] Navegación entre fechas funcional
- [ ] Integración con horarios del profesional
- [ ] Pruebas unitarias creadas
