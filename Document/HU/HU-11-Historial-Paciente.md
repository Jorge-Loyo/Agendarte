# HU-11: Ver Historial de Paciente

## Historia de Usuario
**Como** profesional  
**Quiero** ver el historial de un paciente  
**Para** tener contexto de consultas anteriores

## Criterios de Aceptación
- [ ] El profesional puede acceder al historial desde la agenda
- [ ] Se muestran todas las consultas previas del paciente
- [ ] Se pueden ver notas de consultas anteriores
- [ ] Se muestran datos personales actualizados del paciente
- [ ] Se puede buscar en el historial por fecha o palabra clave
- [ ] Solo se muestra historial de consultas propias del profesional

## Definición de Terminado
- [ ] Vista de historial implementada
- [ ] Listado cronológico de consultas
- [ ] Visualización de notas anteriores
- [ ] Buscador en historial
- [ ] Restricciones de privacidad aplicadas
- [ ] Interfaz intuitiva y responsive
- [ ] Pruebas unitarias creadas
