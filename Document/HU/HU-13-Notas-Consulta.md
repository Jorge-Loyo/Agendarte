# HU-13: Agregar Notas a Consulta

## Historia de Usuario
**Como** profesional  
**Quiero** agregar notas a la consulta  
**Para** registrar información importante del paciente

## Criterios de Aceptación
- [ ] El profesional puede agregar notas durante o después de la consulta
- [ ] Las notas se guardan automáticamente
- [ ] Se puede editar notas de consultas del mismo día
- [ ] Las notas son privadas y solo visibles para el profesional
- [ ] Se registra fecha y hora de creación/modificación de notas
- [ ] Se puede usar formato de texto enriquecido (negrita, cursiva, listas)

## Definición de Terminado
- [ ] Editor de notas implementado
- [ ] Guardado automático funcional
- [ ] Restricciones de edición por tiempo
- [ ] Privacidad de notas garantizada
- [ ] Timestamps de modificación
- [ ] Editor de texto enriquecido
- [ ] Pruebas unitarias creadas
