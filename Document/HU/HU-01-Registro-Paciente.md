# HU-01: Registro de Paciente

## Historia de Usuario
**Como** paciente  
**Quiero** registrarme con mis datos personales  
**Para** acceder al sistema y poder agendar turnos

## Criterios de Aceptación
- [ ] El paciente puede ingresar: nombre, apellido, DNI, edad, sexo, dirección, teléfono, email
- [ ] El sistema valida que el email sea único
- [ ] El sistema valida que el DNI sea único
- [ ] Se requiere contraseña segura (mínimo 8 caracteres)
- [ ] Se envía email de confirmación de registro
- [ ] El usuario queda con rol "patient" por defecto

## Definición de Terminado
- [ ] Formulario de registro funcional
- [ ] Validaciones implementadas
- [ ] Email de confirmación enviado
- [ ] Usuario creado en base de datos
- [ ] Pruebas unitarias creadas
