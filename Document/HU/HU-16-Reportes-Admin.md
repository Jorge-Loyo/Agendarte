# HU-16: Ver Reportes de Turnos y Pagos

## Historia de Usuario
**Como** administrativo  
**Quiero** ver reportes de turnos y pagos  
**Para** analizar el rendimiento del consultorio

## Criterios de Aceptación
- [ ] Se pueden generar reportes por período (día, semana, mes)
- [ ] Se muestran métricas de turnos: agendados, completados, cancelados
- [ ] Se muestran métricas de pagos: ingresos, reembolsos, pendientes
- [ ] Se puede filtrar por profesional específico
- [ ] Los reportes se pueden exportar a PDF o Excel
- [ ] Se muestran gráficos visuales de las métricas

## Definición de Terminado
- [ ] Generador de reportes implementado
- [ ] Múltiples filtros y períodos
- [ ] Métricas completas calculadas
- [ ] Exportación a múltiples formatos
- [ ] Visualizaciones gráficas
- [ ] Interfaz de reportes intuitiva
- [ ] Pruebas unitarias creadas
