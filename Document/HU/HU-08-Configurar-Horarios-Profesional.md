# HU-08: Configurar Horarios de Atención

## Historia de Usuario
**Como** profesional  
**Quiero** configurar mis horarios de atención  
**Para** que los pacientes puedan agendar en mis horarios disponibles

## Criterios de Aceptación
- [ ] El profesional puede definir horarios por día de la semana
- [ ] Se puede configurar hora de inicio y fin por día
- [ ] Se puede definir duración de consultas (30, 45, 60 min)
- [ ] Se pueden configurar descansos entre consultas
- [ ] Se pueden marcar días como no disponibles
- [ ] Se pueden configurar horarios especiales para fechas específicas

## Definición de Terminado
- [ ] Interfaz de configuración de horarios
- [ ] Validaciones de horarios (no solapamiento)
- [ ] Guardado en base de datos
- [ ] Integración con calendario de turnos
- [ ] Vista previa de disponibilidad
- [ ] Pruebas unitarias creadas
