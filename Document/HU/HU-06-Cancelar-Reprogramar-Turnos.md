# HU-06: Cancelar/Reprogramar Turnos

## Historia de Usuario
**Como** paciente  
**Quiero** cancelar o reprogramar turnos  
**Para** gestionar cambios en mi agenda

## Criterios de Aceptación
- [ ] El paciente puede cancelar turnos con al menos 24hs de anticipación
- [ ] El paciente puede reprogramar turnos eligiendo nueva fecha/hora
- [ ] Se muestra política de cancelación antes de confirmar
- [ ] Se procesa reembolso automático si corresponde
- [ ] Se notifica al profesional del cambio
- [ ] Se envía confirmación por email al paciente

## Definición de Terminado
- [ ] Funcionalidad de cancelación implementada
- [ ] Funcionalidad de reprogramación implementada
- [ ] Validaciones de tiempo implementadas
- [ ] Sistema de reembolsos integrado
- [ ] Notificaciones automáticas
- [ ] Pruebas unitarias creadas

## Prioridad: Media
## Estimación: 13 puntos
