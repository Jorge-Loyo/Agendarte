# HU-14: Gestionar Turnos (Administrativo)

## Historia de Usuario
**Como** administrativo  
**Quiero** gestionar turnos de todos los profesionales  
**Para** brindar soporte completo a pacientes y profesionales

## Criterios de Aceptación
- [ ] El administrativo puede ver agenda de todos los profesionales
- [ ] Puede agendar turnos para cualquier profesional
- [ ] Puede cancelar/reprogramar turnos existentes
- [ ] Puede buscar turnos por paciente, profesional o fecha
- [ ] Puede procesar pagos manualmente si es necesario
- [ ] Puede generar reportes de turnos por período

## Definición de Terminado
- [ ] Vista consolidada de todas las agendas
- [ ] Funcionalidades completas de gestión de turnos
- [ ] Buscador avanzado implementado
- [ ] Procesamiento manual de pagos
- [ ] Generador de reportes
- [ ] Interfaz administrativa intuitiva
- [ ] Pruebas unitarias creadas
