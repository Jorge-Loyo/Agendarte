# HU-15: Registrar Nuevos Pacientes

## Historia de Usuario
**Como** administrativo  
**Quiero** registrar nuevos pacientes  
**Para** que puedan acceder al sistema y agendar turnos

## Criterios de Aceptación
- [ ] El administrativo puede crear cuentas de pacientes
- [ ] Se ingresan todos los datos requeridos del paciente
- [ ] Se genera contraseña temporal que el paciente debe cambiar
- [ ] Se envía email con credenciales de acceso al paciente
- [ ] Se puede registrar paciente y agendar turno en el mismo flujo
- [ ] Se valida que no exista duplicado por DNI o email

## Definición de Terminado
- [ ] Formulario de registro administrativo
- [ ] Generación de contraseñas temporales
- [ ] Envío automático de credenciales
- [ ] Flujo integrado registro + agendado
- [ ] Validaciones de duplicados
- [ ] Interfaz administrativa clara
- [ ] Pruebas unitarias creadas
